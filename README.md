## Cómo utilizar este proyecto con Cypress.js

Este proyecto utiliza [Cypress](https://www.cypress.io/) para la automatización de pruebas.

### Requisitos previos

* [Node.js](https://nodejs.org/) instalado en su sistema.

### Instalación

Para instalar las dependencias del proyecto, ejecute el siguiente comando en la terminal:

```bash
npm install
```

### Ejecución de las pruebas

Para abrir la interfaz de Cypress y ejecutar las pruebas de forma interactiva:

```bash
npx cypress open
```

O utilizando el script configurado en el proyecto:

```bash
npm test
```

Para ejecutar las pruebas en modo "headless" (sin interfaz gráfica):

```bash
npx cypress run
```
